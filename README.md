# E-Stockly Manage - FullStack

**E-Stockly** é um gerenciador para controlar estoque e venda de produtos.

### Tecnologias usadas

* Next.JS 14
* Tailwindcss
* Shadcn/ui
* Zod
* React Hook Form
* Prisma
* dayjs
* Postgresql
